# Zimbabwe Health Intelligence Platform (ZHIP)

A Power BI dashboard prototype developed for the **AI4I Proposal**, demonstrating how health data can be transformed into actionable insights for evidence-based decision-making in Zimbabwe.

## Dashboard Journey

The dashboard guides users through a simple three-step process:

1. **Discover** – View national and provincial health performance.
2. **Diagnose** – Identify bottlenecks and root causes.
3. **Act** – Prioritize interventions and monitor progress.

## Requirements

* Microsoft Power BI Desktop (latest version recommended)

## Getting Started

1. Clone or download this repository.
2. Open **`Zimbabwe Health Intelligence Platform_AI4I_Proposal_Design.pbix`** using **Power BI Desktop**.
3. If prompted, enable external content or refresh the data.
4. Explore the report using the navigation buttons, filters, slicers, and drill-through pages.

## Features

* Executive health system overview
* Province and facility performance monitoring
* Digital health readiness analysis
* Resource allocation insights
* Interactive maps, KPIs, and trend analysis

## Intended Users

* Ministry of Health & Child Care
* Provincial Health Offices
* District Health Officers
* Facility Managers
* Development Partners

## Disclaimer

This dashboard is a prototype created for the AI4I proposal. Any sample data included is for demonstration purposes only and does not represent official Ministry of Health records.
